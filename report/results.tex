The test matrix we use is defined as :
\[
A_{ij} =
\begin{cases}
i, i =j \\
0.5, i = j+1, j-1 \\
0.5, (i, j) \in \{(1, n), (n,1)\} \\
0, otherwise
\end{cases}
\]
\[
B_{ij} = 
\begin{cases}
4, i = j \\
-1, i = j+1, j-1 \\
0, otherwize
\end{cases}
\]
